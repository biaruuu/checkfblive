Overview

Base URL: http://localhost:3000  ·  All requests use Content-Type: application/json

EndpointMethodDescription
/api/check-uidPOSTCheck live/dead status of Facebook UIDs
/api/find-facebook-idPOSTFind Facebook ID from URL or username
Check Live UID
POST/api/check-uid
Request Body
FieldTypeRequiredDescription
uidsstring[]YesArray of Facebook UIDs (max 50)
Example Request
{ "uids": ["100012345678901", "100023456789012"] }
Example Response
{
  "results": [
    { "uid": "100012345678901", "live": true  },
    { "uid": "100023456789012", "live": false }
  ]
}

The live field is a boolean — true = live account, false = dead/deactivated.

Find Facebook ID
POST/api/find-facebook-id
Request Body
FieldTypeRequiredDescription
urlstringYesFacebook URL or username to look up
Example Request
{ "url": "https://www.facebook.com/cristiano" }
Example Response
{ "success": true, "uid": "22130498" }

The numeric Facebook ID is in the uid field. success is false when the lookup fails.
