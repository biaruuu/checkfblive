The Check Live UID Facebook tool lets you bulk-verify Facebook User IDs in one click. Paste up to 50 UIDs and instantly see which accounts are live, dead, or errored.

Switch between UID-only, Full, Separated, and Compact view modes to get exactly the output format you need.

What is a Facebook UID?
A unique numeric identifier for every Facebook account — usually 15–16 digits starting with "100".
How many UIDs can I check?
Up to 50 UIDs per batch. Enter one UID per line.
What do Live / Dead / Error mean?
Live = active account. Dead = deactivated or deleted. Error = check failed, try again.
