The Check Live UID Facebook tool lets you bulk-verify Facebook User IDs in one click. Paste up to 50 UIDs and instantly see which accounts are live, dead, or errored.

Switch between UID-only, Full, Separated, and Compact view modes to get the exact output format you need. Copy or export each category independently.

What is a Facebook UID?
A Facebook UID is the unique numeric ID for every account — usually 15–16 digits starting with "100".
How many UIDs can I check at once?
Up to 50 UIDs per batch. Enter one UID per line.
What do Live / Dead / Error mean?
Live = account exists and active. Dead = deactivated or deleted. Error = check failed (try again).
Why do some UIDs show as Error?
Usually rate limiting or network issues. Wait a moment and retry those UIDs.
