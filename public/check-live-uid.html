The Check Live UID Facebook tool lets you bulk-verify Facebook User IDs in one click. Paste up to 50 UIDs and instantly see which accounts are live, dead, or returned an error.

Use the view mode toggles (UID, Full, Separated, Compact) to switch between output formats. Copy or export each result group independently.

What is a Facebook UID?
A Facebook UID (User ID) is a unique numeric identifier for every Facebook account — typically 15–16 digits starting with "100".
How many UIDs can I check at once?
Up to 50 UIDs per batch. Enter one UID per line.
What do Live / Dead / Error mean?
Live = account exists and is active. Dead = account deactivated or deleted. Error = check failed for that UID (try again).
Why do some UIDs show as Error?
Errors usually indicate rate limiting or a temporary network issue. Wait a moment and try again.
