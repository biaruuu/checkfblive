What is a Facebook ID?
Every Facebook account, page, and group has a unique numeric ID (also called UID). This ID never changes even if the username or profile URL changes. It's used to uniquely identify any Facebook entity.
Supported URL Types
Works with profile URLs (facebook.com/username), page URLs, group URLs, post URLs, and plain usernames. Also supports facebook.com/profile.php?id=... format.
How it works
Your request is sent to the server which queries the Facebook ID lookup API. This ensures no CORS issues and provides accurate, reliable results even for private or vanity URLs.
What is a Facebook ID (UID)?
A Facebook ID (UID) is a unique numeric identifier assigned to every Facebook account, page, group, or post. It stays the same even if the person changes their username or profile URL.
How do I find a Facebook ID from a URL?
Simply paste the Facebook profile, page, group, or post URL into the input field and click "Find Facebook ID". The tool will retrieve the numeric ID for you automatically.
Can I find IDs for Facebook pages and groups?
Yes! This tool works with profiles, pages, groups, and posts. Just paste the URL and the tool will extract the correct numeric ID.
Does it work with just a username?
Yes, you can enter just a username (e.g., cristiano ) and the tool will resolve it to the full profile URL and numeric Facebook ID.
Why is the result empty or an error?
This can happen if the profile is private, the URL is invalid, or the upstream API is temporarily unavailable. Make sure the URL is a valid public Facebook page and try again.
