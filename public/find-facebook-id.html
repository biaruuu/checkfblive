The Find Facebook ID tool retrieves the numeric ID behind any Facebook profile, group, page, or post URL. Just paste a URL or username and get the ID instantly.

Use Bulk Mode to look up multiple IDs at once. Export results as a .txt file or copy to clipboard.

What URL types are supported?
Profile, Group, Page, and Post URLs. You can also enter just a username (e.g. cristiano).
Can I look up multiple IDs at once?
Yes — enable Bulk Mode and enter one URL per line.
Why might a lookup fail?
Private profiles, malformed URLs, or rate limiting. Wait a moment and retry.
