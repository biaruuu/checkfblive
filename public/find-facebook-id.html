The Find Facebook ID tool retrieves the numeric ID behind any Facebook profile, group, page, or post. This is useful for API integrations, ad targeting, and data exports.

Enable Bulk Mode to look up multiple URLs at once. Export results as a .txt file or copy directly to clipboard.

What URL types are supported?
Profile, Group, Page, and Post URLs all work. You can also enter just a username.
Can I look up multiple IDs?
Yes! Enable Bulk Mode and enter one URL per line.
What data is returned?
Numeric Facebook ID, display name, account type (profile/group/page), and canonical URL.
Why might a lookup fail?
Private profiles, malformed URLs, or temporary rate limiting can cause failures. Wait a moment and retry.
