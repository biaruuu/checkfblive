The Find Facebook ID tool retrieves the numeric ID behind any Facebook profile, group, or page URL. Paste a URL and get the ID instantly.

Useful for API integrations, ad targeting, and data exports that require numeric Facebook IDs.

What URL types are supported?
Profile, Group, and Page URLs. You can also enter a username directly (e.g. cristiano).
Why might a lookup fail?
Private profiles, malformed URLs, or temporary rate limiting. Wait a moment and retry.
What is the API response format?
The server returns {"success":true,"uid":"12345"} — the numeric UID is in the uid field.
