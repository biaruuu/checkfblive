The Find Facebook ID tool lets you quickly look up the numeric ID behind any Facebook profile, group, page, or post URL. Enter a URL or username and get the ID instantly.

This is useful for developers and marketers who need numeric Facebook IDs for API integrations, ad targeting, or data exports.

Enable Bulk Mode to look up multiple Facebook URLs at once — results can be copied or exported as a text file.

What types of Facebook URLs are supported?
Profile URLs, Group URLs, Page URLs, and Post URLs are all supported. You can also enter just a username (e.g., cristiano).
Can I look up multiple IDs at once?
Yes! Toggle Bulk Mode and enter one URL per line to look up multiple IDs in a single search.
What information is returned?
The tool returns the numeric Facebook ID, display name, account type (profile/group/page), and the canonical URL.
Why might a lookup fail?
Lookups can fail if the profile is private, the URL is malformed, or rate limiting is active. Try again after a short wait.
