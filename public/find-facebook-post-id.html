The Find Facebook Post ID tool extracts the unique numeric ID from any Facebook post URL — whether it's from a profile, group, page, or shared post.

Post IDs are useful for Facebook Graph API calls, ad targeting, embedding posts, and tracking content programmatically.

Powered by id.traodoisub.com — a public extraction API.

What types of post URLs are supported?
Profile posts, group posts, page posts, and shared posts. The URL can be in any standard Facebook format including facebook.com/permalink, /posts/, and /photo/ style links.
Where can I find the post URL?
Click the timestamp of the post (e.g. "3 hours ago") — your browser will navigate to the post's direct URL. You can also tap the three-dot menu on the post and choose "Copy link".
What is a Post ID used for?
Post IDs are needed for Facebook Graph API calls, embedding posts on websites, ad reporting, and any automation that references specific Facebook posts.
Why might a lookup fail?
Private or friends-only posts, malformed URLs, or temporary service issues can cause failures. Make sure the post is publicly accessible and the URL is complete.
